# German-Energy-Shock-Event-Study
This project analyses the impact of the 2022 Ukraine-war-related energy shock on German manufacturing industries using panel data econometrics and event-study methods.

**Dataset Scale**
• 24,000+ monthly observations
• 213 German manufacturing industries (WZ08 4-digit level)
• 2015–2025 panel dataset

**Methods Used:**
• Fixed Effects Panel Regression
• Difference-in-Differences (DiD)
• Event-Study Analysis
• Sector-level Energy Intensity Modelling

**Baseline Model : yit=αi+λt+β(Wart ×EnergyIntensityi)+εit**
	​

****Key Findings: ****
• Energy-intensive industries showed larger post-2022 production declines
• Production effects were stronger than employment effects
• Event-study estimates showed persistent negative sectoral impacts

Data Sources
• Destatis GENESIS
• Eurostat
• World Bank WDI
